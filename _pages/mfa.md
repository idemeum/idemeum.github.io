        <h3 class="display-3">What is Passwordless MFA?</h3>
        <p>Passwords are a big problem - for people who use them and for people who manage them. We reached a point where we are drowning in passwords, and the situation is only getting worse.
        </p>
      </div>
    </div>
    <div class="row">
      <div class="col-lg-6 mx-auto">
        <p><u>Passwordless authentication</u> is the term used to describe a set of authentication methods that don’t rely on passwords or any knowledge-based secret. Biometrics, security keys, and specialized mobile applications are all considered passwordless. Passwordless authentication uses public key cryptography and authenticates the user with a pair of cryptographic keys. It addresses various use cases, including providing access to enterprise applications and devices.</p>
        <p><u>Passwordless MFA</u> is a term used when authentication flow is both passwordless and uses multiple factors, providing the highest security level.</p>

<script>
  $(document).ready(function(){
      /* Get iframe src attribute value i.e. YouTube video url
      and store it in a variable */
      var url = $("#video").attr('src');
      var autoplay = "?autoplay=1"
      var res = url.concat(autoplay)

      /* Assign empty url value to the iframe src attribute when
      modal hide, which stop the video playing */
      $("#modal-video").on('hide.bs.modal', function(){
          $("#video").attr('src', '');
      });

      /* Assign the initially stored url back to the iframe src
      attribute when modal is displayed again */
      $("#modal-video").on('show.bs.modal', function(){
          $("#video").attr('src', res);
      });
  });
</script>
