How to set up idemeum

It only takes 4 simple steps for any user to set up idemeum and start accessing online resources securely.

1. Install app

Download idemeum app from app store and install on a mobile device.

2. Create profile

Add phone number, email address, and initialize biometric authenitcation.

3. Verify identity

Verify identity using goverment issued ID.

4. Access applications

Access applications and online resources from desktop and mobile devices.
